Keybout

Simple multiplayer keyboard racing game

Play

What is it?

KeyBout is a simple web-based multiplayer keyboard racing game, where the players have to type random words before the other players.

The game requires keyboard skills (to type the words as fast as possible), strategy (will you go for the shorter words first, knowing that other players may do the same?) and a bit of luck.

How to play?

First choose a user name then connect to the server. Once connected, you can create or join games. Each game displays random words and users "capture" them by typing their letters. The first player that completely types a word wins it. Whoever wins the most words wins the round.
